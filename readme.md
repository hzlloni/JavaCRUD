이 프로그램은 영어단어장 프로젝트 입니다. 
이 영어단어장을 이용해 추가하고, 파일 불러오고, 저장하고, 검색하고, 수준별로 단어들을 볼까요?

1번을 선택 시, 단어 추가하기
2번을 선택 시, 수준별 단어 보기
3번을 선택 시, 단어 검색하기
4번을 선택 시, 단어가 추가하기
5번을 선택 시, 단어를 수정하고
6번 선택시, 단어 삭제하고
7번 선택시, 추가.삭제.수정 했던 데이터들을 파일에 저장할 수 있습니다 -!!
만약 단어장을 종료하고 싶다면, 0번을 누르시면 종료가 됩니다.
